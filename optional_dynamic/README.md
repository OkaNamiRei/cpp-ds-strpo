# Необязательная часть СТРПО-3: динамические библиотеки

Здесь лежит **заготовка**, а не обязательная часть сдачи. Она нужна, если преподаватель спросит необязательный этап про dynamic libraries.

Внутри:
- `implicit/` — пример shared library для неявного связывания;
- `explicit/` — пример библиотеки с `extern "C" void cat_print(const char*)`;
- `loader/` — пример программы, которая грузит библиотеку во время выполнения.

На текущую обязательную сдачу это не требуется, но как запасной козырь штука полезная. Люди любят дополнительные баллы примерно так же, как дедлайны любят появляться внезапно.

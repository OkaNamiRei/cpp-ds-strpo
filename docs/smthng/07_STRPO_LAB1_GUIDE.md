# 07. СТРПО Лаба 1 — Git и приватный репозиторий

## Что требует актуальное задание

1. Сгенерировать SSH-ключ и привязать его к GitHub.
2. Создать **приватный** репозиторий.
3. Загрузить одну лабораторную по СД.
4. Сделать не меньше **трёх осмысленных коммитов**.
5. Добавить корректный `.gitignore`.
6. Уметь ответить на базовые вопросы по Git.

## Что уже подготовлено в архиве

- `.gitignore`;
- SSH-гайд;
- готовая структура проекта;
- готовые примеры нормальных сообщений коммитов;
- карта, что показывать преподавателю.

## Что остаётся сделать тебе самому

Потому что без твоего GitHub-аккаунта это не материализуется из воздуха:

1. создать приватный репозиторий;
2. залить проект;
3. оформить историю коммитов не одним куском, а нормально;
4. показать это на защите.

## Пример истории коммитов

1. `init: add repository skeleton`
2. `feat(sd1): add rect barrel matrix mystring textwrapper`
3. `feat(sd2): add file hierarchy and codecs`
4. `feat(sd3): add operators db queues boolarray`
5. `build: add cmake presets and tests`
6. `ci: add github actions`
7. `docs: add defense guides`

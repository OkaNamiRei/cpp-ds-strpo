# 12. СТРПО Лаба 6 — трекер задач и Kanban для курсовой

## Самая важная честная вещь

Эту лабораторную **нельзя полноценно закрыть заранее без курсовой**, потому что по актуальным требованиям она должна вестись **в составе курсовой работы**.

То есть сейчас можно сделать только подготовку, а не финальную сдачу.

## Что я уже подготовил

Папка:
- `coursework_prep/`

В ней есть:
- пример разбиения курсовой на issues;
- примеры имён веток;
- формат коммитов;
- шаблон PR;
- схема Kanban-столбцов.

## Что надо будет сделать потом

Когда пришлёшь курсовую:
1. создаём GitHub Project или YouTrack;
2. заводим 5-10 задач;
3. связываем каждую задачу с issue / branch / commit / PR;
4. делаем доску Backlog / In Progress / Testing / Done;
5. держим минимум 3 теста и CI.

## Если используешь GitHub Projects

Формат:
- issue `#2`
- ветка `feature/#2_feature_name`
- commit `[#2] implement ...`
- PR, который закрывает `#2`

## Порядок работы без хаоса

1. Поставить программы по `01_INSTALL_AND_TOOLS.md`.
2. Распаковать архив.
3. Локально собрать и прогнать тесты.
4. Создать приватный репозиторий на GitHub.
5. Сделать первый push по `18_SSH_AND_FIRST_PUSH_STEP_BY_STEP.md`.
6. Разбить работу по веткам и PR по `07_STRPO_LAB1_GUIDE.md` и `08_STRPO_LAB2_GUIDE.md`.
7. Подготовить локальный показ по `03_OPEN_BUILD_RUN_SHOW.md` и `17_DEFENSE_DEMO_SCRIPT.md`.
8. Когда появится курсовая, использовать `coursework_prep/` для СТРПО-6.
9. Если потребуется необязательная лаба 7, открыть `docker/`.

# СД-4: Point, Circle, двухсвязный список окружностей

## Быстрый запуск

```powershell
cmake -S . -B build
cmake --build build --config Debug
ctest --test-dir build -C Debug --output-on-failure
.\build\Debug\lab4_demo.exe
```

## Что здесь реализовано

- `Point`: точка на плоскости.
- `Circle`: окружность с внедрённым объектом `Point`.
- `CircleList`: двухсвязный список копий `Circle`.
- Вставка в начало/конец, удаление первого/всех совпадений, очистка, сортировка по площади, сохранение/загрузка из файла.

## Что открывать на защите

1. `assignment_original/lab4_task4.txt` или `.doc`.
2. `point.*` — точка.
3. `circle.*` — окружность и площадь.
4. `circle_list.*` — список, узлы, вставка/удаление/сортировка.
5. `apps/lab4_demo.cpp` — показ операций.
6. `tests/test_circle_list_lab4.cpp` — проверка.

## Что отвечать

- `Point` внедрён в `Circle`, потому что центр окружности является частью её состояния.
- Узел списка хранит копию `Circle`, чтобы список владел своими данными.
- Двухсвязный список удобен для удаления текущего узла через предыдущий и следующий.
- Сортировка сделана обменом данных, потому что для учебной задачи это проще, чем переставлять связи.

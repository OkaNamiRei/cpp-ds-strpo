# СД-4. Разбор задания по пунктам

## Пункт 1. Создайте класс Point
Реализация: `include/ds/point.hpp`, `src/point.cpp`.

Что открыть:
- объявление класса;
- конструкторы;
- геттеры и сеттеры;
- операторы ввода/вывода.

Что сказать:
- это минимальный класс точки;
- нужен для внедрения в `Circle`;
- хранит координаты центра.

## Пункт 2. Создайте класс Circle
Реализация: `include/ds/circle.hpp`, `src/circle.cpp`.

Что открыть:
- поле `Point center_`;
- список инициализации конструктора;
- метод `area()`.

Что сказать:
- `Circle` содержит внедрённый объект `Point`;
- радиус хранится отдельно;
- конструктор сразу инициализирует `center_` и `radius_`.

## Пункт 3. Разработайте двухсвязный список
Реализация: `include/ds/circle_list.hpp`, `src/circle_list.cpp`.

Что открыть:
- класс `Node`;
- поля `prev`, `next`, `data`;
- фиктивные узлы `head_` и `tail_`;
- конструктор `CircleList()`.

Что сказать:
- список однородный, хранит объекты `Circle`;
- каждый `Node` содержит копию `Circle`;
- `head_` и `tail_` — стражи, упрощающие логику списка.

## Пункт 4. Реализуйте методы класса List
Показывать по очереди:
1. `push_front`
2. `push_back`
3. `remove_first`
4. `remove_all`
5. `clear`
6. `size`, `empty`
7. копирование, перемещение, `swap`

Что сказать:
- вставка идёт созданием нового `Node` между соседями;
- удаление идёт через `delete node`, а деструктор `Node` сам переподключает ссылки;
- `clear` проходит по списку и удаляет все реальные узлы.

## Пункт 5. Сортировка и вывод в файл
Что открыть:
- `sort_by_area`;
- `operator<<` для списка;
- `save_to_file`;
- `load_from_file`.

Что сказать:
- сортировка по возрастанию площади;
- на консоль вывод идёт в удобном читаемом формате;
- в файл вывод идёт в простом формате `x y r`, чтобы его было легко читать обратно.

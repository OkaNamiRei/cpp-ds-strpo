# СТРПО-1: приватный репозиторий, SSH, первые коммиты

Эта папка не обязана собираться как C++ проект: смысл лабы в Git/GitHub.

## Что показать
1. Приватный репозиторий на GitHub.
2. SSH-доступ: `ssh -T git@github.com`.
3. `.gitignore`.
4. Несколько осмысленных коммитов.
5. `git status`, `git log --oneline --graph --all`.

## Что сказать
«В этой лабораторной я показал базовый рабочий цикл Git: локальные изменения, staging, commit, push в приватный репозиторий через SSH. `.gitignore` нужен, чтобы не тащить build, IDE-файлы и бинарники».

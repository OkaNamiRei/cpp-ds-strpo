# Пример разбиения курсовой на задачи

Минимальный набор из 7 задач:

1. Инициализация проекта и базовой структуры.
2. Реализация модели данных.
3. Реализация основной бизнес-логики.
4. Реализация пользовательского интерфейса / CLI.
5. Сохранение / загрузка данных.
6. Автотесты.
7. CI и финальная полировка.

Для GitHub Projects:
- Issue #1 `Project skeleton`
- Issue #2 `Core domain model`
- Issue #3 `Main functionality`
- Issue #4 `Interface layer`
- Issue #5 `Persistence`
- Issue #6 `Tests`
- Issue #7 `CI and release prep`

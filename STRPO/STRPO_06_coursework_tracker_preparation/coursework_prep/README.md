# Подготовка к СТРПО-6 под будущую курсовую

Полностью закрыть СТРПО-6 **невозможно без самой курсовой**, потому что в актуальных требованиях она обязана вестись как часть курсового проекта.

Но можно заранее подготовить всё, что потом просто переносится в рабочий репозиторий:
- шаблон задач;
- Kanban-структуру;
- имена веток;
- формат коммитов;
- шаблон Pull Request.

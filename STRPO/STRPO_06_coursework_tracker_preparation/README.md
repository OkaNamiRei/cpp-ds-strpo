# СТРПО-6: трекер задач для курсовой

По словам преподавателя, эта лабораторная идёт вместе с курсовой. Отдельно без курсовика её лучше не сдавать.

## Что использовать потом
- GitHub Projects или YouTrack.
- Issues под задачи курсовой.
- Ветки от issues.
- Pull Requests с закрытием issues.
- Минимум два показа прогресса: план алгоритма и рабочий код.
